5. Data Security

We implement robust technical and organizational measures to protect your personal information:

  • Encryption: All data is encrypted both in transit (using TLS/SSL) and at rest using industry-standard encryption protocols

While we implement these comprehensive security measures, please note that no method of transmission over the internet is 100% secure.
